# Curriculum

Fast track (do in order, 3-4 h/day, 2-3 days total):

1. [Day 1 — Serving spine](fast_track_day1.md): FastAPI -> Triton -> Postgres -> Redis.
2. [Day 2 — Streaming + observability](fast_track_day2.md): Kafka + ClickHouse + Prometheus/Grafana + OpenSearch.
3. [Day 3 — Spark batch + k8s + polish](fast_track_day3.md).

Then keep grinding on [advanced.md](advanced.md) until the call comes in.

Each day file uses the same shape:

- **Goal** — one sentence.
- **Background** — 5-10 minutes of reading you should do before you touch the keyboard.
- **Exercises** — numbered, each with an explicit *acceptance criterion* so you know when you're done.
- **Talking points** — what to be able to say in 60 seconds about each tool.
