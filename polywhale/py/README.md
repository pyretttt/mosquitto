# Polytop — Python

Textual TUI scaffold for monitoring Polymarket.

## Setup

Requires [uv](https://docs.astral.sh/uv/). From this directory:

```bash
uv sync
```

## Run

```bash
uv run polytop
```

Or:

```bash
uv run python -m polytop
```

## Screens

- **Intro** — hello world landing screen
- **Dashboard** — placeholder for market monitoring
- **Settings** — placeholder for configuration

Press `d` or click **Dashboard** on the intro screen to open the dashboard.  
Press `s` or click **Settings** to open settings.  
Press `Esc` or `b` to go back. Press `q` to quit.
